CSS Algorithm

Cascading

"It is the mechanism in which the browser applies the styles taking into account the origin of the code and order of appearance. Firstly, there is the User Agent or default styles of the browser, secondly the user's customisations and thirdly the styles of the document's author."

Specificity

"It is the weight that a selector has when there is a conflict of styles."
It is calculated as follows:
Labels and pseudoelements = 0,0,0,1
Classes, attributes and pseudo-classes = 0,0,1,0
Id's = 0,1,0,0
Inline Styles = 1,0,0,0
!important = Breaks specificity

Inheritance

"The property of a selector to get or inherit the values of the nearest elements. To apply it, the 'inherit' value is used and to avoid it, the 'initial' value can be used or any other value can be assigned in its absence. "
