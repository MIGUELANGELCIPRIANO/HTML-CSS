Application Tips:

HTML font-size always in pixels.
Pixels: Designs to print.
REM's: Typography, margin and paddings in responsive design.
EM's: Proportional distances between elements.
Percent (%): Width and height of content in responsive design.
Viewport: Cases that take the width and height of the screen as a reference, such as hero image and fluid texts in responsive design.
